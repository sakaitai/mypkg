# ロボットシステム学のROS2のパッケージです


# 時給秒換算システム
このリポジトリは、自分の働いているバイト先の時給を一秒ごとに教えてくれるROS2のノードです。

## 概要
労働時間が8時間を超えたら休憩を知らせるための警告が出ます



# 使用方法
1. このパッケージをセットアップし、依存関係をインストールします。
2. ノードを起動し、

# 実行例
t

# テスト環境
Ubuntu 20.04
